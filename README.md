# Phone Catalog

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Settings

In VSCode press F1, type `Preferences: Open Settings (UI)` paste settings below

```
"eslint.validate": [
    "javascript",
    "javascriptreact",
    "typescript",
    "typescriptreact"
  ],
  "eslint.options": {
    "extensions": [".js", ".jsx", ".ts", ".tsx"]
  }
```
